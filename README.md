# Event Ticketing Platform

An expert-made Event Ticketing & Check-In Platform with Java and Spring Boot.

## Features

- User Registration & Login (JWT Auth)
- Role-Based Authorization (Admin, Organizer, Staff, Attendee)
- Event Management
- Ticket Tiers Management
- Ticket Purchase
- Stripe Payment Integration
- QR Code Ticket Creation
- Email Ticket Sending
- QR Code Check-In
- Live Check-In Dashboard (SSE)
- Waitlist Management

## Technologies

- Java 17
- Spring Boot
- Spring Security
- Spring Data JPA
- PostgreSQL
- Redis
- Stripe API
- ZXing QR Code
- Spring Mail
- Server Sent Events (SSE)
- Gradle

## Architecture

- Monolithic Architecture
- RESTful Design
- DTO Pattern
- Global Exception Handling
- JWT Auth
- Role-Based Access Control
